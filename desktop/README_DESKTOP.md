# ПРОЕКТ ПАПАЖКХ — Desktop (сервер + EXE, офлайн)

## Что это
Папка для сборки автономной версии твоего приложения:
- локальный сервер Flask/Waitress (127.0.0.1)
- UI берётся из ./web (твои текущие HTML/JS/CSS без изменения внешнего вида)
- запускатор launcher.py (GUI) → старт сервера → открыть браузер → кнопка «Остановить»
- упаковка PyInstaller one-folder (без установки Python у конечного пользователя)

## Как собрать на Windows
1) Поставь Python:
   - Windows 7: **Python 3.8.x (64-bit)** (это самый совместимый вариант под Win7)
   - Windows 10: можно тоже 3.8.x, чтобы было одинаково
2) Открой CMD в этой папке и запусти:
   `build_win.bat`

Готовая сборка появится в:
`dist\JKH_Desktop\`

## Как запускать готовую сборку
Запусти:
`JKH.exe`
Откроется окно:
- Запустить
- Открыть в браузере
- Остановить

Данные (в будущем SQLite/бэкапы) будем хранить рядом с программой.
Сейчас твой UI остаётся localStorage-first (ничего не ломаем).
